## About the Project

**Diabetes IQ** is an intelligent diabetes prediction web application designed to help users analyze their diabetes risk using machine learning technology. The platform predicts the possibility of diabetes based on important health parameters such as glucose level, BMI, insulin, blood pressure, age, and more.

The idea behind Diabetes IQ is to spread health awareness and provide users with a quick, simple, and interactive way to understand their health condition through data-driven insights. The project combines healthcare with modern technology to create an accessible and user-friendly prediction system.
